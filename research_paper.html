ReseaRCH Paper


An approach to Opportunistic Resource Management using Java Space in Grid Computing


PEOPLE INVOLVED


Scholar

Fahad Abdel Kader


Supervisor

Shahid Hafeez Mirza


Dated

Start Year : 2004
End Year : 2006


Abstract

As wireless devices such as laptops, tablet PCs etc. are getting faster and faster, the amount of 'idle CPU time' for the average user continues to increase. A typical person usually uses their wireless devices for word processing and other office tasks, listening to music, watching videos and browsing the internet, and thus there will usually be spare 'processing time' available. If this wasted processing capacity of the wireless devices could be aggregated, it could be used to provide processing for computationally intensive applications. This project thesis presents the design, development and evaluation of the prototype framework that uses JavaSpace to meet this goal. The prototype is based on the replicated worker pattern. This allows large, computationally intensive tasks to be divided up into smaller subtasks and then computed by wireless workers. The evaluation of the system indicated that for an application that can be broken into manageable components provides performance gains. Also, the constant monitoring of system state parameters of wireless workers by the resource manager minimizes the intrusiveness to the wireless devices within the wireless grid.
